# Desafio Take Blip - Lista de contatos inteligentes

Esta aplicação refere-se ao desafio técnico proposto pela Take Blip

O projeto esta rodando em uma hospedagem gratuita Netlify, que pode ser conferida em: https://take-blip.thiagoalmeida.com.br;

## Stack do projeto

Esta aplicação front-end foi desenvolvida utilizando as seguintes stacks:
 - Vue.js;
 - Sass;
 - Moment, iMask e outras libs.

## Scripts

Na pasta raiz do diretório, poderá ser executado os seguintes comandos:
 
- `npm i` para instalar as dependências do projeto;
- `npm run dev` para rodar a aplicação em modo de desenvolvimento;
- `npm run build` para realizar o build em produção;

Em modo de desenvolvimento, o projeto esta configurado para executar em http://localhost:8080
